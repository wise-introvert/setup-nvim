### Usage
- Install and setup NeoVIM
    ```bash
    $ sudo chmod +x install.sh
    $ ./install.sh
    ```
- Uninstall and teardown
    ```bash
    $ sudo chmod +x teardown.sh
    $ ./teardown.sh
    ```
